Инструкция, как запустить программный модуль:
1. Установить интерпретатор Python c официального сайта и библиотеку matplotlib.
2. Запустить программу путем вызова команды "python lab2.py".
3. Пример работы программы будет выведен в командную строку. График будет выведен в отдельном окне.
