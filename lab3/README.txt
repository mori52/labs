Для успешного запуска программного модуля следует:
1. Установить интерпретатор Python и необходимые библиотеки.
2. Поместить входные файлы nfcapd_20200.csv, data.csv и саму программу в одну директорию.
3. Запустить программу путем вызова команды  "python lab3.py".
4. После выполнения программы будут созданы файлы list.docx и list.pdf.
